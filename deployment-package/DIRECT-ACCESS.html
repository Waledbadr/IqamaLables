Server Setup Instructions

Windows:

  • For Node.js: Double-click start-app.bat
  • For Python: Double-click start-app-python.bat

Linux/Mac:

  • For Node.js: Run ./start-app.sh
  • For Python: Run python3 server/python-server.py

Access: http://localhost:3000

Web Server Setup

Apache:

  1. Copy app folder contents to /var/www/html/label-printer/
  2. Access via http://your-server/label-printer/

Nginx:

  1. Copy app folder contents to your document root
  2. Configure location block for SPA routing

IIS:

  1. Copy app folder contents to your site folder
  2. Configure URL rewrite for SPA routing
